PALAZZO CINEMA

General Layout

Number of Style Sheets used in total: 4

One extra style sheet present as style.css having all the css in one file.

Style Sheets used:

  • General: used for background color, text font and text color of all pages
  • Main: used to edit the navigation bar and provide structure to the pages.
  • Heading: used to style and align the cinema names
  • Footer: used to style and align the footer and the text in it for all pages
  • Style: Extra style sheet containing all styles in one single sheet.

Classification of Styles:

  1. External stylesheet linked:
    • Used in all pages to help common usage of stylesheets.
    • Provided as a base styling for almost all pages.
  2. Embeded stylesheets:
    • Used to overwrite the base style to help adjust page by page.
    • Used in almost all pages.
  3. Did not use external css stylesheet from website or any inline styling.

Purpose of Styles used:

  1. Home:
    • Making the Home button on the Navigation Menu unclickable.
    • Aligning the thumbnails in the Now Showing and Up Coming section with the text and making the cursor change to a pointer when it hovers on the thumbnails.
    • Making 'Movie Name', 'Director', 'Cast' and 'Duration' bold.
    • Resizing the video and adding a border to it.
    • Styling the headings 'Now Showing' and 'Up Coming'.
    • Aligning the Now Showing and Up Coming sections and adding borders to them and making them into 2 column for a column structure.
    • Aligning the text inside the Now Coming and Up Coming sections to the right of thumbnail.
    • adding backup to the video to ensure all browser support.
  2. Now-Showing:
    • Making the Now Showing button on the Navigation Menu unclickable.
    • Aligning the thumbnails with the form data.
    • Aligning the dropdown menus.
    • Aliging the location images relative to the addresses.
    • Adding border around the whole row consisting of two columns namely the booking section and cinema locations.
    • Styling the headings of the forms.
  3. Ticket:
    • Making the Ticket button on the Navigation Menu unclickable.
    • Making the cells of the seating arrangement table change color as the cursor hovers over them.
    • Aligning the cells of the table of the seating arrangement and adding a border to them.
    • Styling and aligning the table.
    • Styling the text 'Screen', 'Rows', 'Columns' above the table.
    • Styling the heading 'Select your seats'.
    • Adding a border around the form.
    • Aligning the input fields.
    • Forming two columns of ticket details and booking screen.
  4. Print:
    • Adding borders around both of the tickets.
    • Aligning the first ticket.
    • Aligning the second ticket.
    • making them into two columns and giving a border to the row so formed.
  5. Design:
    • Aligning all headings and using padding and spacings.
  6. General:
    • for all pages aligning and adding space, margin and padding them to best fit all browser support.
    • using common themes and ensuring the navigationbar, header and the footer are properly aligned.
    • ensuring no links are broken and following the formate to not underline any when clicked.

Why my HTML code matches the requirements:

  • The code is divided into separate blocks marked by structural tags.
  • JavaScript is not included.
  • Header: Appears at the top of every page and includes a logo and name of the Cinema.
  • Navigator: Follows Header at the top of every page(except Design Page) and includes links to Home, Now Showing and Ticket pages. also ensures that it is unclickable at certain pages.
  • Footer: Appears at the bottom of every page and includes copyright information with links to Home and Design pages.
  • Home Page: A video supporting mp4 and ogg formats is shown and Now Showing and Up Coming sections are visible, each containing three movies and their codes divided into two blocks by div tags.
  • Now Showing Page: A selection menu to select a Cinema and a separate form for each cinema with their respective location details are included.
  • Ticket Page: The details of the selected movie are shown together with the seating arrangement with screen, rows and seats marked.
  • Print Page: The details of two tickets selected (pre-set at this stage) are shown.
  • Ensuring multicolumn set up at all times even when the nav bar is horizontal.
  • The images are properly alligned and padded.
  • when the print page is web preview print it shows on,ly the tickets in seperate pages.

Use of CSS3 Features:

  • Color gradient is used in the cinema company name. (line 42 - general.css file)
  • Border is used with rounded corners in web pages to create a box around content. (Line 72- main.css and also used in embeded style)
  • Imported webfonts from google and font awesome. (line 19- head section of all pages)
  • Use of media queries in print page for editing print preview. (line 50- Print.html)
  • Use of animation for the company logo in all pages. (line 27 onwards- heading.css)
  • Use of CSS to make columns in all pages. (Present in all pages except homepage and also in line 66-main.css)
  • Use of css transform feature to help creat better animation.
  • Use of pseudo class selectors such as :root{} (line 1- general.css), :last-child{}(line 69-print.html)
